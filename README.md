# 📄 SummaXAI

An AI-powered PDF Page Summarizer built using **Streamlit**, **LangChain**, and **Mistral AI**. Upload any PDF, choose a specific page, and instantly generate a concise, structured summary.

---

## 🚀 Features

* 📂 Upload PDF files
* 📖 Preview any page
* 📝 Generate AI-powered summaries
* 🎯 Select specific pages for summarization
* 🤖 Powered by Mistral AI
* ⚡ Clean and responsive Streamlit interface

---

## 🛠️ Tech Stack

* Python
* Streamlit
* LangChain
* Mistral AI
* PyPDF
* Python Dotenv

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/shivansh4565/SummaXAI-ml-project.git
cd SummaXAI-ml-project
```

Install the dependencies:

```bash
pip install -r requirements.txt
```

Create a `.env` file and add your Mistral API key:

```env
MISTRAL_API_KEY=your_api_key_here
```

Run the application:

```bash
streamlit run app.py
```
